## Session outline (5 hrs, trainer-led)

1. **Pre-work + Wk 1 homework debrief** — share architecture docs; group critique of the stack-mapping exercise.
2. **Agent lifecycle + architectures** — design → build → test → deploy → monitor; ReAct, Plan-and-Execute, Reflection, multi-agent orchestration.
3. **Tools — how agents use them** — tool calling, MCP overview; live demo of a Claude agent using tools for a multi-step task.
4. **Memory & state management** — short-term vs long-term, context windows; live demo with persistent Claude Projects.
5. **Build your first agent** — each participant builds v1 of their Week 1 agent spec on Claude; trainer coaches.
