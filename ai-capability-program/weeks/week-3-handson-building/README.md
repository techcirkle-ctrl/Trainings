# Week 3 — Hands-On Agent Building
### Practice, iteration, and depth

**Objective:** Build multiple agents of increasing complexity. Master system prompts, chains, and tool integration.

**Participant artifact:** A working capstone v1 prototype + spec document + colleague feedback notes (carried into Week 4).

## Session outline (5 hrs, trainer-led)

1. **Pre-work + Wk 2 homework debrief** — peer review of defined agents: what worked? what broke? why?
2. **System prompt masterclass** — persona, constraints, knowledge boundaries, output formats, guardrails; rewrite your Wk 2 agent.
3. **Workflows, chains & knowledge** — conditional logic, sequential agents, multi-step prompts; connecting agents to docs, tools, MCP in practice.
4. **Build Sprint 1** — pick 1 of your 3 use cases; full hands-on build; trainer 1:1 coaching; peer review in pairs.
5. **Build Sprint 2 + demos** — build a second agent applying Sprint 1 lessons; each demos a stronger agent (3 min).
