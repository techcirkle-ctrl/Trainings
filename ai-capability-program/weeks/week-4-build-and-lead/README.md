# Week 4 — Build & Lead
### Production-ready agents and leading AI in the org

**Objective:** Build a production-grade agent. Become a functional AI leader who can run org-wide projects.

**Participant artifact:** A production agent + a 30-day deployment plan + a 90-day AI project plan + a clear "what I need from the org."

## Session outline (5 hrs, trainer-led)

1. **Capstone presentations** — each presents prototype (5 min) + group Q&A; trainer identifies the strongest path forward.
2. **Build Sprint: Production-Ready** — refine to production quality: edge cases, guardrails, output quality; pair testing — break each other's agents.
3. **From prototype to organisation** — change management for AI; training your team; measuring impact; org-barrier discussion.
4. **AI project leadership** — scoping, proposing, running AI initiatives; each leader drafts a 1-page proposal for a 90-day AI project.
5. **Final presentations + commitments** — each presents: production agent, 30-day deployment plan, 90-day project, and what they need from the org.
