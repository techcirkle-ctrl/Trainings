# ADR 0001 — Prework is one interactive, self-teaching file per week

**Status:** accepted · **Date:** 2026-06-09 · **Area:** AI Capability Program / `teach` skill

We replaced the original prework model — one HTML file per topic, each a thin "wrapper" that framed why a topic mattered, sent the leader out to an external Google/Anthropic course, and checked comprehension with a quiz, ending in a "bring this to Hour 1" output copied to Teams — with a single self-contained, tabbed, interactive HTML file per week whose sections teach the concepts in-file (Brilliant-style manipulable widgets and step-throughs), track completion locally via `localStorage`, and treat the external courses as optional "go deeper" links. The change was driven by three problems with the old model: (1) it dispatched learners *out* of the material instead of teaching it, so cognitive understanding depended on a course we didn't control; (2) it scattered a week across 5–10 files when leaders want one thing to open and tab through, marking sections complete as they go; and (3) its "flow-back to the orchestrator" framing (copy your output to Teams so Hour 1 can consume it) was wrong for this audience — prework is a *local, self-paced read*, nothing the leader records needs to leave their machine, and the flipped-classroom hand-off is the trainer's job in the room, not a data pipeline. The trade-off: each weekly file is now far larger and more JS-heavy to author (genuine interactions cost real build effort) and the prework no longer mechanically enforces completion — we accept both, betting that interactive self-teaching produces better intuition for a non-engineer leadership cohort than URL-dispatch ever could, and that a committed cohort needs no enforcement gimmick. The decision also requires every weekly prework to mirror its `run-of-show.md` (each major session topic gets a section), which surfaced that prompt engineering and RAG had been taught live with no prework coverage. Reference implementation: `ai-capability-program/weeks/week-1-foundations/00-preread/week-1-prework.html`. Format codified in `.claude/skills/teach/LESSON-FORMAT.md` and `SKILL.md`.
